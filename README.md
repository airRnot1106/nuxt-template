## How to use

```bash
$ git clone git@github.com:airRnot1106/nuxt-template.git
```

```bash
$ cd nuxt-template
```

```bash
$ yarn install
```

## Development

```bash
$ yarn run dev
```

## Build

```bash
$ yarn run build
```
